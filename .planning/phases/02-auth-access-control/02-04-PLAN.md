Build the complete frontend authentication UI: login form, registration form, lock screen, MFA setup/prompt, auth state management hook, and idle timer hook. Wire into App.tsx so the app requires authentication before showing any content.

       ```typescript
       // Auth commands
       registerUser: (input: RegisterInput) =>
         invoke<UserResponse>("register_user", input),
       login: (input: LoginInput) =>
         invoke<LoginResponse>("login", input),
       logout: () => invoke<void>("logout"),
       // Session commands
       lockSession: () => invoke<void>("lock_session"),
       unlockSession: (password: string) =>
         invoke<void>("unlock_session", { password }),
       refreshSession: () => invoke<void>("refresh_session"),
       getSessionState: () => invoke<SessionInfo>("get_session_state"),
       getSessionTimeout: () => invoke<number>("get_session_timeout"),
       // MFA commands
       setupTotp: () => invoke<TotpSetup>("setup_totp"),
       verifyTotpSetup: (secretBase32: string, code: string) =>
         invoke<string>("verify_totp_setup", { secret_base32: secretBase32, code }),
       disableTotp: (password: string) =>
         invoke<void>("disable_totp", { password }),
       checkTotp: (code: string) =>
         invoke<boolean>("check_totp", { code }),
       checkBiometric: () => invoke<BiometricStatus>("check_biometric"),
       enableTouchId: (password: string) =>
         invoke<void>("enable_touch_id", { password }),
       disableTouchId: () => invoke<void>("disable_touch_id"),
       // Break-glass commands
       activateBreakGlass: (reason: string, patientId?: string) =>
         invoke<BreakGlassResponse>("activate_break_glass", { reason, patient_id: patientId ?? null }),
       deactivateBreakGlass: () => invoke<void>("deactivate_break_glass"),
       ```
       Import the new types from "../types/auth".

    4. Create src/hooks/useAuth.ts:
       - Custom hook managing auth state: { user: UserResponse | null, session: SessionInfo | null, isAuthenticated: boolean, isLocked: boolean, loading: boolean, error: string | null }
       - On mount (useEffect), call commands.getSessionState(). If state is "active", also fetch user info. If "unauthenticated", show login.
       - Expose functions: login(username, password), register(username, password, displayName, role), logout(), unlock(password)
       - login: calls commands.login, updates state. If response indicates MFA required (user has totp_enabled), set a "mfaRequired" flag instead of completing login.
       - register: calls commands.registerUser, then auto-login.
       - logout: calls commands.logout, reset state.
       - unlock: calls commands.unlockSession, update session state to active.
       - Also expose: mfaRequired: boolean, verifyMfa(code: string)

    5. Create src/hooks/useIdleTimer.ts:
       - Takes timeoutMinutes param.
       - On mount, add event listeners for mousemove, keydown, click, scroll, touchstart (passive: true).
       - Each event resets a setTimeout timer. When timer fires, call commands.lockSession().
       - On each event, also call commands.refreshSession() (debounced to at most once per 30 seconds to avoid excessive IPC).
       - Cleanup: remove listeners and clear timer on unmount.
       - Return { resetTimer } for manual resets if needed.
